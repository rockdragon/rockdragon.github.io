ポートフォリオ

実際に作成したプロジェクトと、そこから学んだこと

Price Watcher（プライス・ウォッチャー）

2025年12月

Amazon 上の指定商品を対象に、定期的に価格を取得し、価格変動 の履歴をデータベースに保存する。
取得した価格データはフロントエンド上でグラフ表示され、 過去の価格推移を可視化することで、購入タイミングの判断を支援します。

Java SpringBoot Playwright React TypeScript

学び：
Spring Boot における @Scheduled と Playwright を組み合わせた 効率的なデータ取得処理の実装方法を学びました。

AI アシスタント

2026年1月

本システムは音声認識によってユーザーの指示を取得し、意図解析を行います。
その結果を基に、LangGraph と OpenAI API を連携させ、状況に応じた最適なツール選択と出力生成を実現します。

Python FastAPI LangGraph MCP Context Engineering

学び：
このプロジェクトを通じて、LangGraph を用いた Agent ベースのワークフロー設計や、MCP サーバーの呼び出し方法、ならびに基礎的なコンテキストエンジニアリングの実践について学びました。
